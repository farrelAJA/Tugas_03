Belajar Css

Apa Itu CSS?

CSS adalah kepanjangan dari cascading stylesheet, dan berfungsi untuk merapikan html

Bagaimana Cara menggunakan Css?

Ada 3 cara menggunakan css:

  1. Inline Style : menuliskan code langsung pada tag yang akan diubah atau diatur style nya
  2. Internal Style : menuliskan code pada bagian header dokumen html
  3. Eksternal Style : menuliskan code pada file terpisah dan dipanggil melalui tag llink dari dokumen html
